гӮӮгҒЎгӮҚгӮ“гҒҷгҒ№гҒҰгҒ®иЁҖиӘһгҒ«еҪ“гҒҰгҒҜгҒҫгӮӢе®ҡзҫ©гҒ§гҒҜгҒӘгҒ„гҒЁгҒҜжҖқгҒ„гҒҫгҒҷгҒҢгҖҒзҗҶи§ЈеәҰгҒҢй«ҳгҒҫгҒЈгҒҰиүҜгҒӢгҒЈгҒҹгҒ§гҒҷгҖӮ

з’°еўғгғўгғҮгғ«

  • з’°еўғгҒЁгҒҜгҖҒгғ•гғ¬гғјгғ гҒ®й ҶеәҸд»ҳгҒҚйӣҶеҗҲгҒ§гҒӮгӮӢгҖӮ
    • гҒ“гҒ“гҒ§гҖҒгҒӮгӮӢз’°еўғAгҒ®ж¬ЎгҒ«гҒӮгӮӢз’°еўғBгҒҜгҖҒAгҒ®еӨ–еҒҙгҒ®з’°еўғпјҲenclosing environmentпјүгҒ§гҒӮгӮӢгҖӮ
  • гғ•гғ¬гғјгғ гҒЁгҒҜгҖҒжқҹзёӣгҒ®йӣҶеҗҲгҒ§гҒӮгӮӢгҖӮ
  • жқҹзёӣгҒЁгҒҜгҖҒеӨүж•°гҒЁеҖӨгҒ®еҜҫгҒ§гҒӮгӮӢгҖӮ

гҒЁзҗҶи§ЈгҒ—гҒҫгҒ—гҒҹгҖӮ

гҒҫгҒҹгҖҒгӮҜгғӯгғјгӮёгғЈгҒЁгҒҜгҖҒгҖҢжүӢз¶ҡгҒҚпјҲй–ўж•°гҒ®жң¬дҪ“пјүгҖҚгҒЁгҖҢгҒқгӮҢгҒҢи©•дҫЎпјҲдҪңжҲҗпјүгҒ•гӮҢгҒҹз’°еўғгҖҚгҒ®еҜҫгҒ§гҒӮгӮӢгҒЁгӮӮзҗҶи§ЈгҒ—гҒҫгҒ—гҒҹгҖӮгӮҜгғӯгғјгӮёгғЈгҒҜгҒқгҒ®и©•дҫЎжҷӮгҒ«гҖҒгҒқгҒ®жҷӮзӮ№гҒ®з’°еўғгӮ’иҮӘиә«гҒ«дҝқеӯҳгҒ—гҒҫгҒҷгҖӮгҒқгҒ—гҒҰйҒ©з”ЁжҷӮгҒ«гҖҒгҒқгҒ®з’°еўғгӮ’extendsгҒ—гҒҹз’°еўғгӮ’дҪңжҲҗгҒ—гҖҒгҒқгҒ“гҒ§иҮӘиә«гҒ®жүӢз¶ҡгҒҚгӮ’и©•дҫЎгҒ—гҒҫгҒҷгҖӮ

гғ¬гӮӯгӮ·гӮ«гғ«з’°еўғгҒЁгҒӢгӮҜгғӯгғјгӮёгғЈгҒЁгҒӢжқҹзёӣгҒЁгҒӢгҖҒJavaScriptгҒ®ж–Үи„ҲгҒ§гӮӮиҒһгҒҸз”ЁиӘһгҒҢдёҰгӮ“гҒ§гҒ„гҒҰйқўзҷҪгҒӢгҒЈгҒҹгҒ§гҒҷгҖӮгҖҢJavaScriptгҒҜSchemeгҒ®еҪұйҹҝгӮ’еҸ—гҒ‘гҒҰгҒ„гӮӢгҖҚгҒЁгҒ„гҒҶи©ұгӮ’иҒһгҒ„гҒҹгҒ“гҒЁгҒҢгҒӮгҒЈгҒҰгҖҒгҒ“гҒ®иҫәгҒҢгҒқгҒҶгҒӘгҒ®гҒӢгӮӮгҒӘгҖңгҒЁжҖқгҒ„гҒҫгҒ—гҒҹгҖӮ

гҒҫгҒҒгҖҒгҒ“гӮҢгҒҜSchemeгҒ®иӘ¬жҳҺгҒӘгҒ®гҒ§гҖҒJSгҒ®е®ҹйҡӣгҒ®дәӢжғ…гҒҜECMA-262гӮ’еҸӮз…§гҒҷгҒ№гҒҚгҒ§гҒҷгҖӮ

гҒҢгҒЈгҒӨгӮҠиӘӯгӮҒгҒҰгҒҜгҒ„гҒӘгҒ„гҒ®гҒ§гҒҷгҒҢгҖҒдҫӢгҒҲгҒ°10.2 ECMAScript Function ObjectsгӮ’еҸӮз…§гҒҷгӮӢгҒЁгҖҒй–ўж•°гӮӘгғ–гӮёгӮ§гӮҜгғҲгҒ®еҶ…йғЁгӮ№гғӯгғғгғҲгҒ« [[ECMAScriptCode]] гҒЁ [[Environment]] гҒҢгҒӮгӮҠгҖҒгҖҢжүӢз¶ҡгҒҚгҖҚгҒЁгҖҢз’°еўғгҖҚгҒ®еҜҫгҒ«зӣёеҪ“гҒҷгӮӢгӮӮгҒ®гҒҢгҒӮгӮӢгҒЁгӮҸгҒӢгӮҠгҒҫгҒҷгҖӮ

й–ўж•°и©•дҫЎпјҲдҪңжҲҗпјүжҷӮгҒ®жҠҪиұЎж“ҚдҪңгҒ§гҒӮгӮӢOrdinaryFunctionCreateгӮ’иҰӢгӮӢгҒЁгҖҒ 13. Set F.[[Environment]] to env. гҒЁгҒ„гҒҶгӮ№гғҶгғғгғ—гҒҢгҒӮгӮҠгҖҒгҒқгҒ®жҷӮзӮ№гҒ®з’°еўғгӮ’иҮӘиә«гҒ«дҝқеӯҳгҒ—гҒҰгҒ„гӮӢгҒ®гҒҢгӮҸгҒӢгӮҠгҒҫгҒҷгҖӮгҒҫгҒҹгҖҒй–ўж•°е‘јгҒіеҮәгҒ—гҒ®жҠҪиұЎж“ҚдҪңгҒ§гҒӮгӮӢ[[Call]]гҒӢгӮүе‘јгҒ°гӮҢгӮӢPrepareForOrdinaryCallгҒӢгӮүе‘јгҒ°гӮҢгӮӢNewFunctionEnvironmentгҒ§гҒҜгҖҒ 6. Set env.[[OuterEnv]] to F.[[Environment]]. гҒЁгҒ„гҒҶгӮ№гғҶгғғгғ—гҒҢгҒӮгӮҠгҖҒе…ҲзЁӢдҝқеӯҳгҒ—гҒҹз’°еўғгӮ’еӨ–йғЁз’°еўғгҒЁгҒ—гҒҰж–°гҒ—гҒ„з’°еўғгӮ’дҪңжҲҗпјҲextendsпјүгҒ—гҒҰгҒ„гӮӢгҒЁгӮҸгҒӢгӮҠгҒҫгҒҷгҖӮ

гӮ№гғҲгғӘгғјгғ 

SICPгҒ§гҒҜгҖҒгҖҢиҰҒзҙ гҒ®дёҖйғЁгҒҢйҒ…е»¶гӮӘгғ–гӮёгӮ§гӮҜгғҲгҒ§гҒӮгӮӢгғӘгӮ№гғҲгҖҚгӮ’гӮ№гғҲгғӘгғјгғ гҒЁе‘јгӮ“гҒ§гҒ„гҒҫгҒҷгҖӮ

йҒ…е»¶гӮӘгғ–гӮёгӮ§гӮҜгғҲгҒ®гҒҠгҒӢгҒ’гҒ§гҖҒеҝ…иҰҒгҒӘиҰҒзҙ гҒ®гҒҝгҒҢиЁҲз®—гҒ•гӮҢгҖҒз„ЎйҷҗгҒ®иҰҒзҙ гӮ’жҢҒгҒӨгӮ№гғҲгғӘгғјгғ гӮ’иЎЁзҸҫгҒ§гҒҚгӮӢгҒЁгҒ„гҒҶгҒ“гҒЁгӮ’еӯҰгҒігҒҫгҒ—гҒҹгҖӮ

гҒӣгҒЈгҒӢгҒҸгҒӘгҒ®гҒ§гӮ№гғҲгғӘгғјгғ гӮ’JSгҒ§е®ҹиЈ…гҒ—гҒҰгҒҝгҒҫгҒ—гҒҹгҖӮгғ‘гӮәгғ«зҡ„гҒӘйқўзҷҪгҒ•гҒҢгҒӮгӮӢгҒ®гҒ§гҖҒжҡҮгҒӘж–№гҒ гҒ‘иҰӢгҒҰгҒ„гҒҹгҒ гҒ‘гӮҢгҒ°гҒЁжҖқгҒ„гҒҫгҒҷгҖӮ

// йҒ…е»¶гӮӘгғ–гӮёгӮ§гӮҜгғҲгҒҜй–ўж•°гҒ§иЎЁзҸҫгҒҷгӮӢгҒ“гҒЁгҒ«гҒҷгӮӢ

// йҒ…е»¶гӮӘгғ–гӮёгӮ§гӮҜгғҲгҒ®дҪңжҲҗгҒҷгӮӢгҒ«гҒҜж¬ЎгҒ®гӮҲгҒҶгҒ«жӣёгҒҸпјҡ() => и©•дҫЎгӮ’йҒ…е»¶гҒ—гҒҹгҒ„ејҸ

// йҒ…е»¶гӮӘгғ–гӮёгӮ§гӮҜгғҲгӮ’и©•дҫЎгҒҷгӮӢгҖӮйҒ…е»¶гӮӘгғ–гӮёгӮ§гӮҜгғҲгҒ§гҒӘгҒ‘гӮҢгҒ°дҪ•гӮӮгҒ—гҒӘгҒ„
const force = (value) => (typeof value === "function" ? value() : value);

// streamгӮ’ж§ӢзҜүгҒҷгӮӢгҖӮж¬ЎгҒ®гӮҲгҒҶгҒ«ж®ӢгӮҠгҒ®streamгӮ’йҒ…е»¶гҒ•гҒӣгҒҰе‘јгҒіеҮәгҒҷпјҡconsStream(streamгҒ®жңҖеҲқгҒ®иҰҒзҙ , () => ж®ӢгӮҠгҒ®stream)
const consStream = (first, rest) => [first, rest];
// streamгҒ®е…Ҳй ӯгҒ®иҰҒзҙ гӮ’еҫ—гӮӢ
const streamCar = (s) => s[0];
// ж®ӢгӮҠгҒ®streamгӮ’еҫ—гӮӢ
const streamCdr = (s) => force(s[1]);

// mapгҒ®streamзүҲ
const streamMap = (proc, ...streams) =>
  consStream(
    //
    proc(...streams.map(streamCar)),
    () => streamMap(proc, ...streams.map(streamCdr))
  );

// 2гҒӨгҒ®streamгҒ®еҗ„иҰҒзҙ гӮ’и¶ігҒ—гҒҹstreamгӮ’еҫ—гӮӢ
const addStreams = (s1, s2) => streamMap((v1, v2) => v1 + v2, s1, s2);

// streamгӮ’е…Ҳй ӯгҒӢгӮүnеҖӢиЎЁзӨәгҒҷгӮӢ
const printStream = (stream, n) => {
  let s = stream;
  for (let i = 0; i < n; i++) {
    console.log(streamCar(s));
    s = streamCdr(s);
  }
};

// main
const ones = consStream(1, () => ones);
printStream(ones, 5);

  0,
  consStream(1, () => addStreams(streamCdr(fibs), fibs))

гҒ“гӮҢгҒҫгҒ§гҒ®й–ўж•°йҒ©з”ЁгҒҜгҖҒеӨүж•°гҒЁеҖӨгҒ®еҜҫгӮ’жүұгҒҲгҒ°гҒ„гҒ„гҒ®гҒ§з°ЎеҚҳгҒ§гҒ—гҒҹгҖӮдҫӢгҒҲгҒ°гҖҒappend([1, 2], [3, 4])гӮ’йҒ©з”ЁгҒҷгӮӢгҒЁгҒҚгҒҜгҖҒгҖҢlist1гҒЁ[1, 2]гҒ®жқҹзёӣгҖҚгҖҒгҖҢlist2гҒЁ[3, 4]гҒ®жқҹзёӣгҖҚгӮ’дҪңгӮҢгҒ°иүҜгҒ„гҒ§гҒҷгҖӮ

гҒӘгҒ©гҒӘгҒ©гҖҒгҒ„гӮҚгҒ„гӮҚйӣЈгҒ—гҒ„и«–зҗҶгғ—гғӯгӮ°гғ©гғҹгғігӮ°гҒ§гҒҷгҒҢгҖҒгҒқгҒ®гғ‘гғҜгғјгҒҜгҒҷгҒ”гҒ„гӮӮгҒ®гҒ гҒЁж„ҹгҒҳгҒҫгҒ—гҒҹгҖӮPrologгҒҜе…ЁгҒҸгӮҸгҒӢгӮүгҒӘгҒ„гҒ®гҒ§гҒҷгҒҢгҖҒгҒқгҒ®гғЎгӮ«гғӢгӮәгғ гҒ®йҮҚиҰҒгҒӘйғЁеҲҶгҒ«еҚҳдёҖеҢ–гҒҢй–ўгӮҸгҒЈгҒҰгҒ„гҒқгҒҶгҒ гҒЁгҒ„гҒҶж„ҹиҰҡгӮ’жҢҒгҒҰгҒҫгҒ—гҒҹгҖӮ

иҮӘз”ұеӣһзӯ”е•ҸйЎҢ

4з« гҒ«гҒҜе”ҜдёҖгӮ№гӮӯгғғгғ—гҒ—гҒҹз·ҙзҝ’е•ҸйЎҢгҒҢгҒӮгӮҠгҒҫгҒҷгҖӮд»ҘдёӢгҒҜз·ҙзҝ’е•ҸйЎҢ4.79гҒ§гҒҷгҖҒ

з·ҙзҝ’е•ҸйЎҢ 4.79: 4.1зҜҖгҒ§Lispи©•дҫЎеҷЁгӮ’е®ҹиЈ…гҒ—гҒҹгҒЁгҒҚгҖҒеұҖжүҖз’°еўғгӮ’дҪҝгҒЈгҒҰеҲҘгҖ…гҒ®жүӢз¶ҡгҒҚгҒ®еј•ж•°еҗҢеЈ«гҒ§еҗҚеүҚгҒҢиЎқзӘҒгҒ—гҒӘгҒ„гӮҲгҒҶгҒ«гҒҷгӮӢж–№жі•гӮ’еӯҰгӮ“гҒ гҖӮ

пјҲз•Ҙпјү

гӮҜгӮЁгғӘвҫ”иӘһгҒ§гҖҒеӨүж•°гҒ®гғӘгғҚгғјгғ гҒ§гҒҜгҒӘгҒҸгҖҒз’°еўғгӮ’дҪҝгҒҶиҰҸеүҮйҒ©вҪӨгҒ®жүӢжі•гӮ’е®ҹиЈ…гҒӣгӮҲгҖӮгҒқгҒ®з’°еўғж§ӢйҖ гҒ®дёҠгҒ«гҖҒгғ–гғӯгғғгӮҜж§ӢйҖ жүӢз¶ҡгҒҚгҒ®иҰҸеүҮзүҲгҒ®гӮҲгҒҶгҒӘгҖҒеӨ§иҰҸжЁЎгӮ·гӮ№гғҶгғ гӮ’жүұгҒҶгҒҹгӮҒгҒ®гӮҜгӮЁгғӘвҫ”иӘһгҒ®ж§ӢзҜүзү©гӮ’дҪңгӮӢгҒ“гҒЁгҒҜгҒ§гҒҚгӮӢгҒ гӮҚгҒҶгҒӢгҖӮгҒ“гӮҢгӮүгҒ®е•ҸйЎҢгҒЁгҖҒе•ҸйЎҢи§ЈжұәгҒ®жүӢжі•гҒЁгҒ—гҒҰж–Үи„ҲгҒ®дёӯгҒ§жҺЁи«–гӮ’иЎҢгҒҶгҒЁгҒ„гҒҶе•ҸйЎҢпјҲдҫӢгҒҲгҒ°гҖҒ"гӮӮгҒ—PгҒҢзңҹгҒ§гҒӮгӮӢгҒЁд»®е®ҡгҒҷгӮӢгҒЁгҖҒAгҒЁBгҒҢжҺЁи«–гҒ§гҒҚгӮӢ"пјүгҒЁгӮ’й–ўйҖЈгҒҘгҒ‘гӮӢгҒ“гҒЁгҒҜгҒ§гҒҚгӮӢгҒӢгҖӮпјҲгҒ“гҒ®е•ҸйЎҢгҒҜиҮӘз”ұеӣһзӯ”гҒ гҖӮгҒҶгҒҫгҒ„и§Јзӯ”гҒҢгҒ§гҒҚгӮҢгҒ°еҚҡеЈ«еҸ·гҒҢеҸ–гӮҢгӮӢгҒ гӮҚгҒҶгҖӮпјү

гӮ№гӮӯгғғгғ—гҒ—гҒҹгҒ®гҒҜгҖҢгҒ“гӮҢгӮүгҒ®е•ҸйЎҢгҒЁгҖҒе•ҸйЎҢи§ЈжұәгҒ®жүӢжі•гҒЁгҒ—гҒҰж–Үи„ҲгҒ®дёӯгҒ§жҺЁи«–гӮ’иЎҢгҒҶгҒЁгҒ„гҒҶе•ҸйЎҢпјҲдҫӢгҒҲгҒ°гҖҒ"гӮӮгҒ—PгҒҢзңҹгҒ§гҒӮгӮӢгҒЁд»®е®ҡгҒҷгӮӢгҒЁгҖҒAгҒЁBгҒҢжҺЁи«–гҒ§гҒҚгӮӢ"пјүгҒЁгӮ’й–ўйҖЈгҒҘгҒ‘гӮӢгҒ“гҒЁгҒҜгҒ§гҒҚгӮӢгҒӢгҖӮгҖҚгҒ®йғЁеҲҶгҒ§гҖҒиҮӘз”ұеӣһзӯ”гҒЁгҒ•гӮҢгҒҰгҒ„гҒҫгҒҷгҖӮ

жӯЈзӣҙгҖҒдҪ•гӮ’е•ҸгӮҸгӮҢгҒҰгҒ„гӮӢгҒ®гҒӢгӮӮгӮҸгҒӢгӮҠгҒҫгҒӣгӮ“гҖӮжүӢгӮӮи¶ігӮӮгҒ§гҒҫгҒӣгӮ“гҖӮгҒҠгҒқгӮүгҒҸгҖҒе…ҲзЁӢжӣёгҒ„гҒҹгҖҢзөҗи«–гҒӢгӮүд»®е®ҡгӮ’жҺЁи«–гҒ—гҒҰгҒ„гӮӢгӮҲгҒҶгҖҚгҒҢй–ўйҖЈгҒ—гҒҰгҒ„гӮӢж°—гҒҜгҒҷгӮӢгҒ®гҒ§гҒҷгҒҢгҖҒгҖҢеҚҡеЈ«еҸ·гӮ’гҒЁгӮҠгҒҹгҒҸгҒӘгҒЈгҒҹгӮүгҒҫгҒҹиҖғгҒҲгӮӢгҒ“гҒЁгҒ«гҒҷгӮӢгҖҚгҒЁгҒ„гҒҶгҒөгҒ–гҒ‘гҒҹгӮігғЎгғігғҲгҒ§гҒҠиҢ¶гӮ’гҒҫгҒңгҒҫгҒңгҒ—гҒҰгҒҠгҒҚгҒҫгҒ—гҒҹгҖӮ

5з« 

д»®жғізҡ„гҒӘгғ¬гӮёгӮ№гӮҝгғһгӮ·гғігҒ®гӮЁгғҹгғҘгғ¬гғјгӮҝгӮ’е®ҹиЈ…гҒ—гҖҒгҒқгҒ®ж©ҹжў°иӘһгҒ®гҒҶгҒҲгҒ§4з« гҒ®гғЎгӮҝеҫӘз’°и©•дҫЎеҷЁгӮ’е®ҹиЈ…гҒ—гҒҫгҒ—гҒҹгҖӮ
