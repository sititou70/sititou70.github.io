「キューブの基礎」入門

「解けないキューブ」の説明をするために，まずはキューブの基本構造を学びましょう．

覚えることは3つだけ

1. サブキューブの種類3つ

ルービックキューブは，小さなキューブが集まって出来ています．

正式な用語ではありませんが，本記事ではこれらを サブキューブ と呼びましょう．

それぞれのサブキューブは， 色の数（露出する面の数） によって3つに分類できます．

  1. 左にあるのは，色数が1色のサブキューブです．ルービックキューブの，各面の真ん中に位置します．そのため センターキューブ と呼ばれます．
  2. 真ん中にあるのは，2色のサブキューブです．ルービックキューブの辺に位置します．
  3. 右にあるのは3色のサブキューブです．ルービックキューブの角に位置します．

簡単ですね．

2. サブキューブの配色は変わらない

ここに，黄色 + 赤色のサブキューブがあります．

当たり前ですが，ルービックキューブをどのように操作しても，この黄色と赤色が離れてバラバラになることは絶対にありません．1つのサブキューブですから， 常にセットです．

パカッとやらない限りはね．

3. センターキューブの位置関係は変わらない

ルービックキューブをどのように操作しても，センターキューブの位置関係を変えることは出来ません．

ルービックキューブの，可能なすべての動きを見てみましょう．

センターキューブの位置関係は変わりません．ルービックキューブでは， センターキューブを中心に他のサブキューブが動くだけ なのです．

「解けないキューブ」の見分け方

ここまでくれば，ほぼ勝利です．おめでとうございました．

さて，解けないキューブを見分けるコツは， 隣接する面の色 です．

当たり前ですが，ルービックキューブの特定の面は，4つの面に隣接します．例えば…

この動画では，緑面は，赤面 / 黄面 / オレンジ面 / 白面の 4色の面 に隣接しています．3次元空間にある立方体ですからあたりまえ体操です．

そして，キューブをどのように操作しても，これらの 「隣接する色の組み合わせ」は変わりません ．例えば，先程のキューブでは，青面と緑面はそれぞれ反対にあり，隣接していませんでした．このキューブで遊んでいるうちに「いつの間にか緑面と青面が隣接していた」ということは絶対にあり得ません．なぜなら，センターキューブの位置関係は固定ですし，青と緑が隣接するようなサブキューブは最初から無いし生まれません．そらそうだ．

では，あらためて ↓ 問題のキューブを見てみましょう．

↓ 解説画像です

青色に注目して2色 / 3色のサブキューブを見ると（解説画像上段），このキューブでは，青面は白面 / 赤面 / オレンジ面 / 緑面に隣接していることがわかります．

加えて，センターキューブの位置関係に注目すると（解説画像下段），青面は黄面とも隣接しているとわかります．

青面が白 / 赤 / オレンジ / 緑 / 黄の 5面に隣接している！？ これは矛盾です． したがって，このキューブは解けません！

少なくとも，3次元空間にあるルービックキューブではないので，この女の子は高次元存在ということになります．

しかし，このキューブも 解けません

このキューブの配色が普通じゃない可能性もあるだろ

との指摘があるかもしれません．青面同士，緑面同士，赤面同士の隣接があるということは，このキューブの配色は，青面 / 青面 / 緑面 / 緑面 / 赤面 / 赤面となり，3色のみということになります．

実は，3色だけのキューブというのは，本当に存在します．

実は，このキューブも 解けません

長い試練も終わり，皆さんは無事 解けないキューブ鑑定士免許皆伝 となったわけですが，注意すべきことが1つあります．

解けないキューブを見つけたとしても，それを無闇に指摘してはいけません．

例えば，ある絵のキューブが矛盾しているからといって，そのPixivのコメント欄に

TPOをわきまえるのじゃぞ．師匠との約束じゃ．

その先へ

鑑定士に飽き足らず 解けないキューブ仙人 になりたいという，そこのあなたに朗報です．実は，配色に矛盾が無いからといって，そのキューブが「解ける」とは言えないのです．そこらへんの理論を極めたいなら群論を勉強してみてはいかがでしょうか．【群論】とは，代数学におけるめっちゃ重要なやつです（うまくお茶を濁す）．以下は，群論のWikipediaのスクリーンショットです．
